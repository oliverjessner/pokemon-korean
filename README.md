# pokemon-korean 
I got inspired by [sindresorhus](https://github.com/sindresorhus/pokemon)
    
## Install

```
$ npm install --save pokemon-korean 
```

## Usage

```js
const pokemon = require('pokemon-korean ');
console.log(pokemon[0]); // Isanghaessi
```
